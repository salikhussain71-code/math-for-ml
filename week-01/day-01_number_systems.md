# Week 1 Day 1 - Number Systems

## Topic
Integers, Rational Numbers, Irrational Numbers

## Why this matters
This is the foundation for ML, AI, and Data Science. If this is strong, algebra and coding become easy.

## 1. Natural Numbers (N)
Numbers for counting: 1, 2, 3, 4, 5...
Not natural: 0, -5, 1.5, √2

## 2. Whole Numbers (W)
Natural numbers + 0: 0, 1, 2, 3, 4...
Example: 0, 7, 100

## 3. Integers (Z)
All positive, negative, and zero. No decimals or fractions.
...-3, -2, -1, 0, 1, 2, 3...

**Rules:**
- + × + = +
- - × - = +
- + × - = -
- - × + = -

Example: -5 × -2 = 10

## 4. Rational Numbers (Q)
Any number you can write as p/q where q ≠ 0.

Examples:
- Fractions: 1/2, 3/4
- Integers: 5 = 5/1
- Terminating decimals: 0.5, 2.75
- Repeating decimals: 0.333..., 0.1212...

## 5. Irrational Numbers
Cannot be written as p/q. Decimal never ends, never repeats.
Examples: √2, π = 3.1415..., e = 2.718...

Rule: If √x is not a whole number, it's irrational.

## 6. Real Numbers (R)
All rational + irrational numbers together.

## 7. Quick Practice
1. Is 0 an integer? → Yes
2. Is 1/2 rational? → Yes  
3. Is √2 rational? → No
4. Is π irrational? → Yes
5. Is √49 rational? → Yes, because √49 = 7

## 8. Common Mistake
Not all decimals are irrational. 0.5 is rational.
Not all roots are irrational. √64 = 8 is rational.
Every integer is rational, but not every rational is integer.
